== Description ==

The plugin checks the text of comments when they are added and, if it detects obscene (profan, foul) words, replaces them with the word you specified. You can add your own forbidden words. Latin and Cyrillic are supported.

= Main features =

* Replacing profan words with your chosen word.
* Checking comments when adding. Previously added comments will not be corrected.
* Available settings: own bad words list, replacement word and its color.
* You can replace foul words with one word, signs, a whole sentence, or just a space.
* You can also add new obscene words in WP dashboard.
* Plugin translated into languages: English, Russian, Ukrainian.

== Installation ==
1. Upload plugin folder to the /wp-content/plugins/ directory. OR download plugin in .zip format and install it in WordPress.
1. Activate the plugin through the ‘Plugins’ menu in WordPress.
1. Go to "Settings => SpeedPlus AntiMat" and customize it.

== Frequently Asked Questions ==

= Comments in what language can this plugin check? =

The plugin processes comments in Cyrillic and Latin. Therefore, the plugin effectively works with swear comments in English, Russian, Ukrainian, Belarusian and other languages.
